# Password Generator

A simple Password Generator built with React.

This project generates a random password based on the selected length and options chosen by the user.

## Features

- Generate random passwords
- Change password length using a slider
- Include numbers in the password
- Include special characters in the password
- Password updates automatically when options change

While building this project, I practiced and learned:

- React Functional Components
- `useState` for managing state
- `useEffect` for running code when state changes
- Handling input events with `onChange`
- Working with range inputs (sliders)
- Working with checkboxes
- Generating random values using JavaScript
- Updating the UI dynamically based on user interaction

## How It Works

1. The user selects a password length using the slider.
2. The user can choose whether to include:
   - Numbers
   - Special Characters
3. Whenever any option changes, a new password is generated automatically.
4. The generated password is displayed on the screen.
